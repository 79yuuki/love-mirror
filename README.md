# LOVE MIRROR

BaPA product

## Quick Start

Start love mirror.

Should install "editorconfig" & "jsHint" to your editor.

```
cd love-mirror
npm install
node app.js
```
and open http://localhost:5000


## URLs

###  '/'

PC page

### '/shake'

login page or shake page

### '/auth/facebook'

facebook login

### '/auth/facebook/callback'

callback url and redirect to 'shake'
